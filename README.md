# Welcome to our data science project: The Librarian
# Recommending media for politics and computer science topics!

### Authors: Manas Goswami & Prakrati Kadekar

## How to use:

### Python Necessery Libraries:
- pandas
- sentence_transformers
- re (built-in)
- numpy
- kaggle
- time (built-in)
- os (built-in)
- python-dotenv
- google-api-python-client

### Other Necessery Downloads:
- npm

### Running the Project:
1. Open up 2 terminals
2. In the first terminal run python main.py
3. In the second one, cd into the librarian-ui directory
4. Then in the second terminal run npm install
5. Then run npm run dev
6. Wait for both terminals to finish loading. The first terminal should show that debugger is active. The second terminal should show a local host that you can visit.
7. Then load the local host website on your browser. Now you can search up a political or tech topic you want to learn about
